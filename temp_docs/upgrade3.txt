# 5) RADAR SCORING ENGINE (DETERMINISTIC)
Create file:
`src/lib/radar/scoring.ts`

## Scoring formula:
signal_score = clamp(
  source.priority
  + freshness_boost
  + title_quality_boost
  + keyword_boost
  + noise_penalty,
  0,
  100
)

### Freshness boost:
- <12h +30
- <24h +25
- <48h +18
- <7d +10
- else 0

### Keyword boosts:
- SEO hits *15
- AI hits *12
- Engineering hits *10

### Noise penalties:
- roundup -15
- podcast/webinar -10
- sponsored -20
- summary <120 chars -10

Also implement:
- `shouldDisplay(score)`
- `shouldFeature(score)`
